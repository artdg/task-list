# Task List (React + Vite)

Небольшой таск‑лист для ежедневных задач с приятным современным UI и сохранением данных в `localStorage`.

## Возможности

- Добавление задач (Enter — добавить)
- Приоритет: высокий / средний / низкий
- Дедлайн (опционально)
- Поиск по задачам
- Фильтры: все / активные / готовые
- Массовые действия: отметить всё, снять отметки, очистить выполненные, очистить всё
- Редактирование по двойному клику (Enter — сохранить, Escape — отмена)
- Переключение темы (сохранение выбора)

## Запуск

```bash
npm install
npm run dev
```

Сборка:

```bash
npm run build
npm run preview
```
